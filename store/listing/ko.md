## 설명 (Description)
진짜 무료입니다 — 워터마크, 시간제한, 구독, 광고가 없습니다. 녹화부터 프레임
편집, 최고 화질 GIF 내보내기까지 15MB 남짓의 가벼운 앱 하나로 끝내세요.
계정도, 로그인도, 인터넷 연결도 필요 없습니다.

녹화는 캡처 프레임 창으로 합니다. 무엇이 녹화되는지 정확히 보여주는, 옮기고
크기를 조절할 수 있는 창이죠. 크기와 위치를 기억하므로 다음 캡처를 곧바로
시작할 수 있고, 전체 화면은 체크박스 하나로 전환됩니다. 최대 60fps로
부드럽게 캡처합니다.

녹화를 멈추면 모든 프레임이 편집기로 들어옵니다. 필요 없는 프레임을 지우고,
타이밍과 재생 속도를 조정하고, 자르고, 크기를 바꾸고, 무엇이든 되돌리세요.
시작과 끝의 멈춰 있는 구간은 버튼 하나로 정리되고, 화면이 바뀌지 않은 중복
프레임을 병합해 용량을 크게 줄일 수 있습니다.

한 장면을 망쳤다면 처음부터 다시 찍을 필요가 없습니다. 이어서 녹화로 그 부분만
다시 캡처해 원하는 위치(맨 앞, 현재 프레임 뒤, 맨 끝)에 끼워 넣으세요. 새로
찍은 테이크는 색깔 블록으로 자동 그룹화되어 통째로 옮길 수 있습니다 — VoidGif의
멀티 테이크 편집입니다.

내보내기에서 진가가 드러납니다. GIF는 gifski 엔진으로 프레임마다 수천 가지 색을
살려 밴딩 없이 인코딩되며, 무손실 APNG, PNG 시퀀스, H.264 MP4도 지원합니다.
내보내기 전에 예상 용량을 미리 보여주고, GitHub·Discord·Slack 프리셋을 고르면
용량 제한에 맞춰 자동으로 최적화합니다. 완성되면 클립보드로 바로 복사해
붙여넣으세요.

작업은 자동 저장됩니다. 앱이 예기치 않게 종료되어도 다음 실행 때 복구할 수
있습니다. 그리고 모든 것이 사용자 기기 안에서만 이루어집니다 — VoidGif는
네트워크에 연결하지 않고 어떤 데이터도 수집하지 않습니다.

주요 기능
- 크기·위치를 기억하는 캡처 프레임으로 영역/전체 화면 녹화, 최대 60fps
- 멀티 테이크 편집: 이어서 녹화 + 삽입 위치 지정 + 색깔 블록 그룹 이동
- 프레임 편집: 삭제, 복제, 순서 변경, 프레임별 딜레이, 자르기, 크기 조절, 실행 취소
- 기존 GIF 파일 열어서 편집 — 자르고, 타이밍 바꾸고, 용량 줄여 다시 내보내기
- 정지 구간 자동 정리, 중복 프레임 병합으로 용량 절감
- 재생 속도 조절(전체/선택/그룹별), 루프 다듬기(왕복 재생, 이음새 미리보기)
- 내보내기 전 예상 용량 표시 + GitHub/Discord/Slack/X 용량 프리셋 자동 맞춤
- gifski 기반 고화질 GIF, 무손실 APNG, PNG 시퀀스, H.264 MP4
- 내보내기 후 클립보드 복사로 바로 붙여넣기
- 자동 저장과 비정상 종료 복구
- 다크/라이트 테마, 한국어·English·日本語
- 전역 단축키: F7 시작/일시정지, F8 정지
- 완전한 오프라인. 계정·광고·추적·데이터 수집 없음
